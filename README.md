# DSA-CA1-Smart-E-Bike-Docking-Station-App
An app for users around the docklands to suggest areas to install E-Bike docking systems that securely lock bikes and charge them. This app will allow users to add an area in need, users can also take the area with the highest priority and add it to a ongoing job stack. There will also be a linked list where users can add areas to be evaluated.
